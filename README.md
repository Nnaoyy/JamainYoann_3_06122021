# Openclassroom Projet 3 :Dynamisez une page web avec des animations CSS


##### Vous venez d’être recruté chez Ohmyfood!, en tant que développeur junior. Félicitations !

_Ohmyfood!_ est une jeune startup qui voudrait s'imposer sur le marché de la restauration. L'objectif est de développer un site 100% mobile qui répertorie les menus de restaurants gastronomiques. En plus des systèmes classiques de réservation, les clients pourront composer le menu de leur repas pour que les plats soient prêts à leur arrivée. Finis, les temps d'attente au restaurant !


### Compétences évaluées :
* Assurer la cohérence graphique d'un site web
* Utiliser un système de gestion de versions pour le suivi du projet et son hébergement
* Mettre en œuvre des effets CSS graphiques avancés
* Mettre en place son environnement Front-End
* Mettre en place une structure de navigation pour un site web

## Objectifs
Objectifs  
● Phase 1 : Développer un site proposant le menu de 4 grands restaurants parisiens.  
● Phase 2 : Permettre la réservation en ligne et la composition de menus.
